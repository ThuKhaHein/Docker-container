🎯 Usage Guide

Navigation

  1. Click Dashboard to view KPIs and charts
  2. Click Fuel Inventory to see all records with filtering
  3. Click Add Entry to record new fuel transactions
  4. Click Equipment to view equipment inventory

Adding Fuel Records

  1. Navigate to Add Entry tab
  2. Select transaction date
  3. Choose type: "Fuel Received" or "Fuel Used"
  4. If receiving, price field will appear automatically
  5. Enter quantity in liters
  6. Provide source/destination name
  7. Add optional notes
  8. Click Add Entry or Clear

Filtering Inventory

  1. Navigate to Fuel Inventory tab
  2. Select filters: Type, Start Date, End Date
  3. Click Filter to apply
  4. Click Reset to clear all filters

Exporting Data

  • 📄 PDF: Click Export PDF button - generates formatted report
  • 📊 Excel: Click Export Excel button - creates .xlsx file
  • Both exports include all filtered/visible data

Editing Records

  1. In Fuel Inventory, find your record
  2. Click the ✏️ Edit icon
  3. Modal dialog appears with pre-filled form
  4. Update any fields as needed
  5. Click Save Changes or Cancel

Deleting Records

  1. In Fuel Inventory, find your record
  2. Click the 🗑️ Delete icon
  3. Confirmation modal appears
  4. Click Delete to confirm or Cancel to abort

🐛 Troubleshooting

Charts not displaying?

Ensure your database has sufficient data. Charts need labels and data points. Check browser console for errors.

Modal doesn't close?

Try refreshing the page or clearing browser cache. Ensure JavaScript is enabled. Check for console errors.

Export buttons not working?

Verify that jsPDF and XLSX libraries loaded correctly. Check console for "undefined" errors. Ensure table has data.

Form fields not validating?

Check HTML5 input types (date, number, required). Ensure JavaScript form handlers are attached. Test in different browser.
